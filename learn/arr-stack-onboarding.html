🔒 Built-in Privacy & Security

✅

VPN Kill Switch

If the VPN connection drops, all download traffic is instantly blocked. Your real IP can never leak.

✅

Encrypted Tunnel

WireGuard encryption wraps all download traffic. Your ISP sees encrypted data going to a VPN server — nothing more.

✅

Network Isolation

Only the download clients use the VPN. Your other apps connect normally — no speed penalty for browsing or streaming.

✅

No Port Forwarding Needed

Everything runs on your local network. No services are exposed to the internet unless you choose to.

✅

SSL Usenet Downloads

Usenet traffic is encrypted end-to-end via SSL. Even inside the VPN tunnel, the data is double-encrypted.

✅

No-Log VPN Provider

Providers like ProtonVPN and Mullvad keep zero logs. Even they can't see what you download.

🌍

Stream Anywhere — Not Just at Home

With a free service like Tailscale, you can stream your media from anywhere — at work, on vacation, at a friend's house. Just download the app on your phone and on your server, and you're done. All it does is trick your phone into thinking it's still on your home Wi-Fi, so you can keep streaming off your server like you never left. No complicated setup, no subscriptions.

🚀 Ready to Set It Up?

Everything you need is included. Here's the game plan at a high level — the detailed guide walks you through each step.

1

Read the guide

The included arr-stack-guide.md is the full walkthrough. Open it in something that renders Markdown nicely (VS Code, Obsidian, a browser extension) so it's easy to follow. Even better — feed the guide, compose file, and .env file to an agentic AI coding assistant like Claude Code, Cursor, or Windsurf. These tools can read the guide, edit config files, run commands, and automate most of the setup for you — turning hours of manual work into a guided, semi-automated process.

2

Run the compose file

The included compose.yaml is ready to go. A single docker compose up -d spins up the entire automation stack.

3

Pay for the services you want

Some services require subscriptions — a VPN at minimum, and optionally a Usenet provider and indexer. Sign up for whichever ones make sense for you, then plug the credentials into the relevant apps.

4

Connect everything with API keys

The arr apps need to talk to each other. The guide covers this in detail, but the gist is: generate an API key in each app's settings and paste it where the other apps ask for it.

5

Set up your media player apps

The compose file handles the automation pipeline, but it doesn't include the apps you actually watch, listen, and read with — things like Jellyfin, Kavita, Audiobookshelf, or a music player. You'll set those up separately and point them at the media folders the stack organizes. The diagram above shows some popular choices.
